User Management Information
User Roles
  • Admin: Has full access to all features, including user management and webhook token management.
  • User: Can connect to IBKR, view trading information, but cannot manage users or webhook tokens.
Security Best Practices
  • Create individual accounts for each user to maintain accountability.
  • Limit the number of admin accounts to only those who need administrative access.
  • Regularly review user accounts and remove any that are no longer needed.
  • Ensure all users have strong, unique passwords.
